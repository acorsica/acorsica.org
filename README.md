# acorsica.org

Public website for **C.O.R.S.I.C.A.** — *Corse Organisant la Réunion Sur Internet de Compétences Autonomes*.

This repository is the human-facing entry point for the association website. It is intentionally separate from:

- [`acorsica/gouvernance`](https://github.com/acorsica/gouvernance), the governance and official-document archive;
- [`acorsica/.github`](https://github.com/acorsica/.github), the GitHub organization profile;
- initiative repositories such as [`acorsica/privai`](https://github.com/acorsica/privai), [`acorsica/Kudos`](https://github.com/acorsica/Kudos), and [`acorsica/marianivillage`](https://github.com/acorsica/marianivillage).

## Role

`acorsica.org` is the public doorway. The legal, documentary, and governance source of truth remains in `acorsica/gouvernance`.
